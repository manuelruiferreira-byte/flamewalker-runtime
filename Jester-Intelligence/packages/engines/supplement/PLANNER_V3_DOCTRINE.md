# ACE Mind Supplement Planner v3

The planner separates five authorities:

1. Body permission and interaction conflicts are hard gates.
2. Actual tick history is the only evidence of real intake.
3. Frequency debt, maximums, minimum gaps, and residual windows govern repetition.
4. Astrology, BaZi, numerology, and Mayan signals produce an explicit 0/4 to 4/4 convergence band.
5. The planetary hour controls timing for already-permitted items. It never overrides a body hold, cooldown, maximum, or conflict.

Future-date browsing uses a temporary virtual schedule. It does not write future recommendations into actual intake history.
