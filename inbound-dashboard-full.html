TRI is required for any active gate or open floor location. A truck may carry multiple TRIs, so the TRI field accepts values like TRI-021270 + TRI-021271. When a row is partially filled, missing planning fields light red and the Data column shows what is required.
Receive mode matters: report column Type Trsp Inbound seeds AUTOMAT = Automatic and MANUAL = Manual. Mixed means one truck has automatic and manual TRI flows together. Manual or Mixed unload creates a receive/verification task. Automatic unload moves directly to received + putaway required.
Load type matters: Container = detached container, needs shift-out, and implies restack. Truck/Container = container-type load on truck/trailer, no shift-out, no restack.
Contract SLA active: Arrival time is stamped when the user/AI marks a gate busy or unloading, unless a manual arrival time is provided. Truck / Truck/Container arriving on time must depart within 2 hours of actual arrival. If not on time, it may wait longer but should stay visible as flexible. Detached containers on gate must be unloaded maximum 1 day after arrival.
